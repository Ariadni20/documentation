
This directory contains various polarization consistent (PC) basis sets.

| pc-n     | The original PC basis sets                   |
|          | [J. Chem. Phys. 115, 9113 (2001)]            |
|----------+----------------------------------------------|
| aug-pc-n | Same as pc-n but with augmentation functions |
|          | [J. Chem. Phys. 115, 9113 (2001)]            |
|----------+----------------------------------------------|
| pcJ-n    | PC optimized for J-couplings                 |
|          | J. Chem. Theory Comput. 2, 1360 (2006)       |
|----------+----------------------------------------------|
| u-pcJ-n  | Uncontracted pcJ-n                           |
|          | J. Chem. Theory Comput. 2, 1360 (2006)       |
|----------+----------------------------------------------|
| pcS-n    | PC optimized for NMR shieldings              |
|          | J. Chem. Theory Comput. 4, 719 (2008)        |
|----------+----------------------------------------------|
